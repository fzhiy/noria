# Security Policy

## Supported Versions

Only `0.x` releases currently receive security updates.

| Version | Supported |
| ------- | --------- |
| `0.x`   | Yes       |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, report them privately by email to `noreply@noria.dev` with:

- A clear description of the issue
- Affected version or commit information
- Steps to reproduce, if available
- Any known impact or exploitation details

You will receive an acknowledgment as soon as practical. After triage, the
maintainers will assess severity, work on a fix, and coordinate disclosure when
appropriate.
