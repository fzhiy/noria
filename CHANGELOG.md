# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project aims to follow [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-04-15

### Added

- Initial public release of NORIA.
- Inbox approval gate for controlled intake of new materials.
- MCP evidence service for provenance-aware retrieval workflows.
- Copybara sync pipeline for private-to-public publication.
- Core tooling surface spanning 30 tools.
- Operational workflow surface spanning 25 commands.
